## <div dir="rtl">הרצה על הסביבה שלך</div>
<p dir="rtl">יש להוסיף משתנה סביבה DB_URL שיכיל את ה-URL של ה-MongoDB. על כן יש להוסיף קובץ .env לנתיב ההרצה של הפרויקט ולהגדיר אותו כך:</p>

```
DB_URL=<your MongoDB url>
```
